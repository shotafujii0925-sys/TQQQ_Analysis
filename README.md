# TQQQ vs S&P500: 統計的有意性とレバレッジ・リスクの検証

## プロジェクト概要
本プロジェクトでは、少額資産からの早期リッチ化を目的としたレバレッジ戦略（TQQQ）の有効性を、統計学の観点から検証しました。直近3ヶ月の市場データを用い、マクロ指標との相関分析および対応のあるt検定を実施しています。

## 検証結果の要約
- **統計的有意性の欠如:** TQQQのリターンは表面上SPYを上回りましたが、t検定の結果（p値 ≈ 0.54）、その差は統計的に有意ではなく、単なる高ボラティリティによる「ノイズ」である可能性が高いことが示されました。
- **リスク調整後リターンの劣後:** シャープレシオの比較において、レバレッジによるボラティリティ増幅がリターンの寄与を上回り、投資効率が1倍指数（SPY）に対して低下していることを確認しました。
- **最大下落率（Max Drawdown）:** 短期間で約33%のドローダウンを記録。数学的な「回復の非対称性」により、長期保有におけるリスクが浮き彫りとなりました。

## 結論
データに基づくと、TQQQの優位性は特定の強気相場に依存しており、統計的・数理的な観点からは、一貫してS&P500をアウトパフォームする根拠は認められませんでした。リスク管理を優先する観点から、安定した複利効果を期待できる1倍インデックスが合理的であると結論付けます。
